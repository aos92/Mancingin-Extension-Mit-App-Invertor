Mancingan Suite

Kumpulan extension MIT App Inventor yang dirancang khusus untuk membangun aplikasi manajemen lomba mancing (galatama) — dari database SQLite, renderer denah lapak, manajemen menu kantin, hingga komponen tabel data dinamis.

Pilih Extension

Mancingin

Core engine SQLite untuk manajemen perlombaan galatama. Mencakup CRUD generik, transaksi, klasemen live, buku kas, dan blok domain lomba mancing tingkat tinggi.

Buka Dokumentasi

LapakCanvas

Renderer denah lapak kolam berbasis Canvas. Menampilkan posisi lapak dengan status kosong/terisi secara visual, lengkap dengan interaksi klik per lapak.

Buka Dokumentasi

MenuGalatama

Manajemen item kantin dan transaksi makanan selama lomba. Mendukung operasi CRUD menu, pemilihan item, dan integrasi langsung dengan database galatama.

Buka Dokumentasi

TableGalatama

Auto-renderer tabel data hasil query database dengan tampilan yang bisa dikustomisasi. Cocok untuk menampilkan klasemen, daftar peserta, dan laporan transaksi.
